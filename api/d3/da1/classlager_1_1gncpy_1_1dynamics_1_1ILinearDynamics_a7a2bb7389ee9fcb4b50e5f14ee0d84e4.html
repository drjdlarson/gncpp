Get the State matrix.

This is the main function child classes will need to implement as it defines the dynamics for the class. This is allowed to be time varying and depend on external parameters. These parameters may be subclassed to allow specific values per dynamics type. This returns the A matrix from the following equation.

\[ x_{k+1} = F(t) x_k + G(t) u_k \]
